# Ocean Cleanup App - Clean Demo Walkthrough

This script is designed for a clear, step-by-step presentation. 
**Style:** Direct, pacing-focused, and feature-highlighting.

| **Visual Action** | **Voice Script** |
| :--- | :--- |

| | |
